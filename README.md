
This is a course project for **SIT305**, developed by **Haoyu Liu**.

The current progress of the project is the implementation of a Travel Companion App that supports essential travel-related conversions.

So far, the app can convert:
- Currency
- Travel units
- Temperature

The project is developed in **Android Studio** with **Java** and **XML**.
